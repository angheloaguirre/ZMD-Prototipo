# **ZMD E-commerce Prototype**

## **Descripción del Proyecto**
Este proyecto es un prototipo de la tienda en línea para **ZMD**, una microempresa peruana especializada en la reparación de dispositivos electrónicos y la venta de accesorios para tecnología. El prototipo se centra en ofrecer una experiencia digital de compra y servicio técnico de manera sencilla, eficiente y atractiva para el usuario.

Enlace al prototipo desplegado en Microsoft Azure: https://calm-bush-0c7b8f70f.3.azurestaticapps.net/

## **Objetivos del Proyecto**
- **Digitalización de ZMD**: Facilitar el proceso de compra de accesorios electrónicos y la gestión de servicios de reparación a través de una plataforma web.
- **Expandir el Alcance**: Ampliar el mercado de ZMD más allá de su local físico mediante el uso de comercio electrónico.
- **Mejorar la Experiencia del Cliente**: Ofrecer un diseño intuitivo y responsivo, asegurando una experiencia de compra fluida y segura.

## **Características Principales**
- **Página de inicio**: Presenta un diseño limpio con acceso a las categorías de productos y servicios.
- **Catálogo de productos**: Los usuarios pueden explorar una variedad de accesorios electrónicos, con opciones de filtro y búsqueda.
- **Página de producto**: Detalles completos de cada producto, incluyendo imágenes, descripciones, precios y opciones de compra.
- **Carrito de compras**: Funcionalidad que permite agregar productos, ver el total y proceder al pago.
- **Pasarela de pago**: Integración con opciones de pago como Niubiz, Yape y Plin.
- **Diseño responsive**: Optimizado para dispositivos móviles y escritorio, asegurando accesibilidad en cualquier plataforma.

## **Tecnologías Utilizadas**
- **Frontend**: HTML, CSS, JavaScript (React)
- **Backend**: Node.js, Express
- **Base de Datos**: MongoDB
- **Pasarela de pago**: Niubiz (para pagos digitales)
- **Otras Herramientas**: Figma (para diseño del prototipo)

## **Estructura del Proyecto**
- **/src**: Código fuente del prototipo (HTML, CSS, JS)
  - **/components**: Componentes reutilizables (botones, formularios, etc.)
  - **/pages**: Páginas principales (inicio, catálogo, detalle de productos)
  - **/styles**: Archivos de estilos CSS
  - **/assets**: Imágenes y recursos visuales
- **/docs**: Documentación adicional y referencias
